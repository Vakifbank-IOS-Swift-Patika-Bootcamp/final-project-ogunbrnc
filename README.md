# First Opening of the Application

• When the application is first opened, permission is requested to send notifications.

<img src="https://user-images.githubusercontent.com/48843277/208312043-8f39483c-32f9-4ff6-b0c7-4a90d0104d7c.png" width="256" height="512">

# 1.Tab - Games:

## Game List Screen:

• When the screen is first turned on, indicator is shown until the games are loaded.

• All games are sorted by popularity first when games are loaded.

• 20 games are shown each time, pagination is provided. New games are loaded as the screen is pulled down.

• You can search for games using the search bar. As the screen is pulled down, new games related to the relevant search are loaded.

Home Screen | Home Screen Search

• You can go to the details by clicking on the game.

## Game Detail Screen

• On the game detail screen, the game's picture, name, duration, number of comments, average rating and rating numbers from green to red are displayed.

• Just below are the platforms, related tags, related genres and game description.

• By clicking the favorite button in the upper right, we can add the game to the favorite and remove it from the favorite.

<img src="https://user-images.githubusercontent.com/48843277/208311652-2736d369-3038-4ebb-9024-7f150c0955ec.png" width="256" height="512">

# 2.Tab - Favorite Games:

• If there is no game in the favourite, a message is displayed to warn the user.

• When the game is added to favorites on the detail page, it is directly displayed in the favorite list.

• We can delete the game from the favorite list, when we delete it, if the game is open at that moment on the detail page, it will appear directly that it has been removed from the favorite.
